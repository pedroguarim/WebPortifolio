# 🌐 Web Portfolio

Meu portfólio pessoal desenvolvido com foco em apresentar meus projetos, habilidades e evolução como desenvolvedor.

## ✨ Sobre o projeto

Este projeto foi criado com o objetivo de consolidar meus conhecimentos em desenvolvimento web, aplicando boas práticas de HTML, CSS e JavaScript.

Além disso, o portfólio serve como uma vitrine dos meus projetos acadêmicos e pessoais.

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* Responsividade (Mobile First)
* Integração com APIs (quando aplicável)

## 🎯 Funcionalidades

* 🌍 Suporte a múltiplos idiomas (PT/EN)
* 📱 Layout responsivo
* 🎨 Animações e transições suaves
* 📂 Seção de projetos com links para GitHub
* 📧 Botão de contato com cópia de e-mail
* 🔗 Links para redes sociais
